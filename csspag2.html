Div e Pseudos Elementos

A div basicamente faz uma "preenchimento" da área que ela representa, seja por definições css ou tamanho de texto, vídeos, áudio e etc. Quando você quiser dar uma configuração dentro da div por exemplo um paragrafo, basta utilizar o seletor "div > p {}", ele basicamente está dizendo que que o div tem um filho que é um paragrafo.
Os pseudos elementos servem para adicionar conteúdo no css, seja nos id ou classes você o expressa colocando a teg que quer mudar e dois pontos(a::), exemplo se você quere adicionar uma seta no seu link basta colocar a::before{content: '→';}, assim você adiciona uma seta, você mudar o fundo a cor da letra e entre outras muitas coisas.

Tipos de caixas

tipos de caixas, nos temos os box-level, quando criado ele criar uma linha nova e ocupa toda a LARGURA da tela e quando criado boxs desse mesmo tipo ele pula para a linha de baixo, outra característica é que ele quebra as linhas quando colocado, a teg associado á ele é a <div>.
A Inline-level eles são criados dentro da linha selecionada e diferente do box-level ele não ocupa 100% da largura da tela ocupando somente a largura do conteúdo nele colocado, outra característica é que ele não quebra a linha quando colocado assim sendo feito na mesma linha a teg associada á ele é a <span> A teg <span> sereve para marca com uma caixo um trecho do texto.

Tegs border padding margin

Margin serve para deixar as caixas longe umas das outras. todas as caixas tem configurações de posicionamento.

Tegs outline ouline

outilne serve como contorno das bordas podendo adicionar contornos personalizados ou importados. O o ouline serva para adicionar uma linhas traçejadas entre outros tipos de linhas.

bordas personalizadas

Você pode criar suas próprias bordas basta pegar uma imagem de borda que você fez e colocar o seguinte código:

border-image-source: url('borda.png'); border-image-slice: 27; border-image-repeat: repeat;

Em border-image-source você seleciona a opção url e nela basta colocar entre aspas o nome do arquivo de imagem que está sua borda, em border-image-slice: você seleciona o tamanho da sua imagem e em border-image-repeat: você seleciona a opção repeat que basicamente vai repetir sua imagem várias vezes. Aqui tembém temos como simplificar o código basta colocar na sequência que foi demonstrada no exemplo acima, exemplo:

border-image: url('borda.png') 27 repeat;

Variáveis e transition-duration

Para colocar variáveis em css basta colocar a teg :root, quando for definir uma variável tem que colocar dois traços na frente, você pode fazer váriaveis de cores fontes entre outros, você precisa definir uma nome para definir a variável, exemplo:

  • --cores0:#c5ebd6;
  • --cores1:#83e1ad;
  • --cores2:#83e1ad;
  • --cores4:#1a5c37;
  • --cores5:#063d1e;

No exemplo usamos cores toda vez que colocar em cores e arrastar para baixo vai aparecer todas as variáveis, no caso precise mudar todas as cores do site basta trocar a cor da variável que ele mudará todas as configurações que tenha aquela variável, isso serve para fontes também.

A teg transition-duration serve para definir o tempo de transição quando você passa o mouse em cima, isso serve para botões interativos links em geral qualquer coisa que você queira mudar o tempo de animação.

Configuraões do Background img

Na Teg background img você pode adicionar as imagens que quiser basta clicar control espaço que ele dará o caminho para chegar na pasta, você também pode colocar background url e colocar o link de uma imagem que achar na web. A teg background-size define o tamanho das imagens, ele também tem duas opões que é o auto que vem como padrão o contenti que sempre tenta exibir a imagem todo mesmo que crie espaços em branco nas laterais ou em cima, muito visto em vídeos, e também o cover que vai fazer a imagem ocupar a tela toda mesmo que tenha que cortar alguns pedaços.
A teg background-repeat tem várias configurações, de caso coloque a opção no-repeat ele não repitirá as imagens independente do tamanho, a opção reapeat x e y, o x repetirá a imagem indo para os lados e o y repetirá as imagens para baixo e para cima.
A teg background-position serve para posicionar onde você quer que a a imagem fique, sendo elas as posições emcima embaixo centro, podendo usar duas direções ao mesmo tempo como direita emcima esquerda embaixo entre outros.

Construção de Tabela simples

Todas as tabelas simples em html segue essa seguinte hierarquia primeiro <table >dentro de table temos o table row que significa linha de tabela, dentro do table row temos também o table header cabeçalho de tabla e table data que é os dados da tabela. O table row tem como teg <tr> e o tabole header tem como teg <th> e o table data tem como teg <td>.
Para colocar bordar na sua tabela você utiliza a teg border nos seus dados (Teg <td>), podeno definir como exemplo 1px solid black, para deixar as linhas das bordas juntas basta utilizar a teg border-collapse e utilizar a opção colapse, os fechamentos dessas tegs não é obrigatório Para alinhar sua table pode ser usado a teg text-aling para alinhar horizontalmente e para alinhar verticalmente utilizamos a teg vertical-aling depois só usar o alinhamento que desejar, um ponto para se lembrar que para alinhar ao centro dessa teg não é utilizando center e sim o opção middle.

Construção de Tabela Grandes

Anatomia para tabelas grandes, dentro do table temos em primeiro o <caption> que é a legenda da tabela temos também o <thead> dentro dessa teg podemos ter tr td th, também temos o <tbody⋗ dentro dessa teg podemos ter tr td th e o < tfoot> que podemos ter dentro dele a seguintes tegs ter tr td th.
Se você quiser indicar que o th que você fez é o título da coluna utilizamos dentro da teg <th scope="col"> essa opção indica que é o título daquela coluna, se você quer indicar que aquele th é o título da linha basta utilizar a teg <th scope="row"> isso indica que é o título daquela linha. Se de casso seu título abranja mais de uma coluna utulizamos <th scope="colgruos"> e se ele ocupar mais de uma linha utilizamos <th scope="row">, mas isso só vale se os dados que ele reverencia estar em mais de uma linha ou colunas.

Efeito Zebrado

Para fazer um efeito zebrado na sua tabela utilizamos a pseudo-classe :nth-child() dentro da parênteses colocamos em quantas linhas ele vai pular, basta colocar 2n, 3n ,4n e assim por diante, também pode definir que as linhas seram puladas em números impar ou pares utilizando oddo para impar e even para par, não esquece de colocar a cor de fundo para fazer o efeito, claro essa pseudo-classe não faz só isso ele faz uma formatação pulando as linhas determinadas, basta que aja um cenário para utiliza-la de outros formas.
Outra coisa a se atentar que se quisermos que a tabela tenha um efeito zebrado de branco e preto, colocaremos o branco com impar e o par como preto por exemplo, ai que está uma confusão o seu site virá como padrão a cor de fundo branco de caso você não defina a cor de fundo porque já está branco, quando decidir colocar uma cor diferente de branco, sua tabela ficará dessa cor, porque se não definimos a cor da tabela ela tomará a cor do site como base, então lembre de definir a cor da tabela.

Fixar Cabeçalho e mesclar células

Para fixar seu cabeçalho primeiro temos que deixar o posicionamento da tabela relativo depois criamos a seguinte sequência thead >tr >th {} dentro do coxete colocamos o position sticky que é para deixar grudado e o top:0; isso significa que ela vai colocar na parte 0, vale lembrar que essa configuração não funciona em todos os navegadores.
Para fazer que uma célula ocupe duas colunas utilizamos dentro da teg td a sequinte opção <td colspan="2"> o dois indica a quantidades de colunas que ele ocupará se quiser que ele ocupe mais colunas basta aumentar o valor. Para fazer que a célula ocupe mais de uma linha utilizamos a seguinte teg <td rowspan="3"> lembrando quando você for comer outra linha você tem que exclui-la ou já fazer a montagem da tabela pensando nas células que vão ocupar determinados números de linhas e colunas.

Cor de Colunas e Rolagem Lateral

Aprendemos como configurar e editar as linhas, mas se eu quiser por exemplo deixar as colunas uma de cada cor? para isso basta utilizar a teg <colgrup> nela colocamos as tegs <col> onde colocamos as mesmas quantidade de colunas que temos na nossa tabela, nisso podemos definir diferentes classes ou id para cada coluna, e editar no css colocando (col.nome da classe aqui), mas de casso eu querer que uma classe ou id muda duas colunas de uma vez para isso do lado da classe colocamos a teg <span="2"> onde o número é a quantidade de colunas que queremos editar.
Quando sua tabela é grande de mais ela cria uma rolagem lateral para resolver isso envelopamos sua tabela com uma div, coloque id o class do nome que quiser e vamos utilizar a teg css overflow, essa teg configura todo o conteúdo que transborda do seu site, nela utilizaremos o overflow-x que configura a rolagem horizontal , nela selecionaremos a opção auto e pronto nessa teg temos outras configurações mas nesse casso já é o suficiente.
