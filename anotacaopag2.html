Teg strong e em

A teg<strong> ser para colocar o texto em negrito,exemplo:casa

<strong>casa</strong>

.

Já <EM> serva para colocar o texto em itálico,exemplo: casa

<em>casa</em>

Teg mark

A teg <mark> serva para marca o texto igual a caneta marcadora,exemplo:casa

<mark>casa</mark>

Teg big e samall

A teg <big> deixa o texto grande mas está obsoleta a teg, exemplo: casa

<big>casa</big>

Já a teg <small> serve para deixar o texto pequeno.

<small></small>

Teg start

A teg <start=""> é uma teg que define de onde vai começar o número ou a letra da sua listas, sendo letra ou número a teg start sempre define onde começa por números mesmo o type sendo letra ou número romano,exemplo se você define sua lista e quer que começe pela letra D você coloca no start o número 4 porque a letra D é a quarta letra do alfabeto, exemplo:

  1. fui buscar pão
  2. Tomar café
  3. Ir para a escola
Velá como o código é:

<ol start="4" type="A">
<li>fui buscar pão</li>
<li>Tomar café</li>
<li>Ir para a escola</li>

Teg dl dt e dd

A teg <dl> é para criar um litas de definições onde dentro dela usamos a teg <dt> que serve para definir o termo que será usado na lista, também temos a deg <dd> que serva para escrever a definição do termo que usamos na lista,exemplo:

Termo utilizado:HTML
Definição do termo:Linguagem de marcação para a crição de um site
Termo utilizado:CSS
Definição do termo:Linguagem de marcação para a criação do desing de um site
Termo utilizado:JAVASCRIPT
Definição do termo:Linguagem de progamação para a criação de interatividade de um site
Vejá como o código é:

  • <dl>
  • <dt>Termo utilizado:HTML</dt>
  • <dd>Definição do termo:Linguagem de marcação para a crição de um site</dd>
  • <dt>Termo utilizado:CSS</dt>
  • <dd>Definição do termo:Linguagem de marcação para a criação do desing de um site</dd>
  • <dt>Termo utilizado:JAVASCRIPT</dt>
  • <dd>Definição do termo:Linguagem de progamação para a criação de interatividade de um site</dd>

Link externo

A teg <a> serve para você colocar links de site externo e interno, para esse link não sobrepor seu site você faz o seguinte:

<a href="" target="_blank" rel="externo">

Em <href=""> você coloca o link, em target="" você coloca "_blank" para que ele tenha um alvo em uma janela em branco e em <rel="externo"> você coloca externo para indicar que aquele é um link externo.

Link intreno

Para anexar um link intreno você utiliza a teg <a> mas a página que você criar tem que tá na mesma página do seu index.html se no caso estiver em uma pasta basta colocar .// para voltar uma pasta.
Quando criar links interno é sempre bom colocar na teg <a> o <rel="next"≫ indicando que é a próxima página, se for voltando uma página coloque prev,exemplo:

<a href="exemplo" rel="next" target="-self">próxima páginma</a> <a href="exemplo" rel="prev" target="-self">página anteriot</a>

Colocando next para indicar próxima página e prev para indicar página anterior. Também colocamos em <target="-self"> para indicar que esse link é no mesmo site.

link de empresa

Quando você for colocar um link de alguma expressa que você não é patrocinado você coloca no rel="nofollow" para indicar que essa empresa não tem nada a ver com você.

<a href=link da empresa" rel="nofollow"></a>

Voltar pasta

Para voltar uma pasta você utiliza ../ isso indica que você está voltando,exemplo:

<a href=".//exemplo/link que você quer"></a>

link para download

Para colocar download no seu site você usa a teg <a> coloca o link para download, escreva лdownload=""> colocando o nome do arquivo que será baixado e também é necessario colocar tipo do arquivo,colacando type="o tipo de arquivo" e coloca o o tipo do arquivo utilizando o link que está no site links vejá o exemplo:

<a href="link do aquivo para download" download="nome do arquivo" ></a>
