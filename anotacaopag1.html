Código base

Para fazer um código base basta criar um index.html e colocar o sinal de !. Dentro desse código base temos as seguintes tegs:
A teg <html lang> é a teg que define a língua do site, que no caso é português.

<htm lang="pt-br">

Teg Meta Charset="UTF-8

A teg <meta charset="UTF-8"> esse teg basicamente faz você poder ter acentuação no site, utilizando o código:UTF-8

<meta charset="UTF-8">

Teg Head

A teg <head> dentro dessa teg você pode fazer várias configurações do seu site entre outras coisas.

<head>

Teg Body

A teg <body> é o conteúdo do seu site, é onde vão ficar os parágrafos títulos vídeos áudio entre outras coisas.

<body>

Teg Meta Name="" Content=""

A teg <meta name="" content=""> em resumo faz seu site ocupar 100% da tela.

meta name="viewport" content="width=device-width, initial-scale=1.0">

Teg Title

A teg <title>Document</title> é o titulo do site.

<title>Document</title>

Teg H1

A teg <h1> é onde vocé vai colocar o título do seu conteúdo.

<h1></h1>

Teg P

A teg <p>é o parágrafo de seu site onde você colocará os textos.

<p> </p>

Teg Hr

<hr>é para criar um linha horizontal.

<hr></hr>

UL

A teg <& l t;> coloca o sinal <, e a teg <& g t;> coloca o sinal >.

<& l t;> <& g t;>

URL

A URL é o endereço de um site, exemplo:www.google.com. é nela onde podemos achar os sites.

Hierarquia dos Títulos

A hierarquia dos títulos <H1> <H2> <H3> e assim por diante são níveis de títulos, exemplo:<H1>principal receitas da vovó. <H2> receitas doces, <H2> é subtítulo de <H1>, <H3>doce de amendoim, <H3> é subtítulo de <H2> e asssim por diante. você pode ter vários subtítulos, e títulos também, só toma cuidado para não exagerar.
Não existe que você só pode ter um<H1> isso é mentira, oque você tem que ter em mente é que <H1> é o título principal da sua página.

Teg img

A teg <img>serve para adicionar imagens, quando clicar nessa teg aparecerá o seguinte código:<img src="" alt=""> em img src="" você colocará a imagem que você quer, para facilitar sua vida, basta usar o atalho clique+espaço entre as aspas que dará as opções de arquivos de imagens disponíveis. Em alt="" você dará o nome da imagem, isso serva para quando não carregar a imagens falar oque era aquela imagem.

<img src="" alt="">

Teg code e teg pre

A teg <code> deixa as letras com o memso espaçamento que está no visualcode.

<code> </code>

Teg Pre

A teg <pre> deixa o código do jeito que está no visulcode nó site.

<pre> </pre>

Teg <Q> e <Blockquote>

Quando houver uma citação de uma frase de alguém utilizamos a teg <q> para colocar a frase entre as aspas, isso serve para o mecanismo de busca do google ver que é uma citação. Podemos também usar a teg <blockquote> para escrever um citação de um frase de alguém, nessa citação podemos colocar o link de onde tiramos esse citação, exemplo:<blockquote cite="aqui colocamos o link de onde tiramos a citação, seja de um texto ou um livro"> aqui no meio da abertura e fechamento colocamos a frase citada</blockquote"> Assim além do google ver que é uma citação ele vê de on de veio essa citação.
Veja como é o Código

<blockquote cite=""></blockquote>

Teg <abbr>

A teg <abbr> serve para você colocar o significado da apreviação/sigla, exemplo:
HTML<abbr title="Aqui colocamos que significa a apreviação, no caso de html significa Hypertext Markup Linguage"> No meio da abertura e fechamento da teg colocamos a apreviação que no caso é HTML <abbr> O resultado final vai ficar assm: HTML
Veja como fica o código:

<abbr> title="Hypertext Markup Linguage">HTML</abbr>

Teg <bdo>

A teg <bdo> só inverte o texto, veja exemplo <bdo dir="Aqui colocamos a opção: rtl">Aqui colocamos a frase que queromos inverter, exemplo: Fui para casa</bdo> O resultado vai ficar assim: Fui para casa
Veja como é o Código

<bdo> dir="rtl" Fui para casa</bdo>

Teg <ol>

A teg <ol> serve para criar listas ordenada com uma sequência de números ou letras, podemos colocar o tipo da sequncia utilizando a colocando type:<ol type=" Tendo como opções essas: 1,A,a,I,i"> Cada uma dessas opções coloca um tipo de sequência diferente.
Veja como o código é:

<ol> type=""</ol>

Teg <ul>

A teg <ul> serva para criar listas não ordenadas que não segue uma sequência, para trocar o tipo de icone que aparece ultilizamos type
<ul type="Aqui selecionamos o tipo de icone que queremso, sendo eles: disc,circle,square">.
Veja como o código é:

<ul> type=""</ul>

Teg <li>

A teg <li> serva para colocar oque queremos na nossa lista, seja ela ordenada ou não, exemplo: <li>Buscar pão <li>Tomar café <li>Ir para a ecola. A teg <li> não precisa de um fechamento mesmo o visualcode colacando um. A teg <li> é coloca dentro da teg <ol> ou <ul> essa lista ficaria assim utilizando a teg <ol> ficaria assim:

  1. fui buscar pão
  2. Tomar café
  3. Ir para a escola
Vejá como o código é:

<ol type="1"> <lifui buscar pão</li> <liTomar café</li> <liIr para a escola</li> </ol>
