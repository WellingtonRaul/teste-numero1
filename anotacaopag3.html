Teg img

Para colocar imagens no site você utiliza a teg dentro da teg você coloca a teg <img> coloca a imagem em <src="imagem escolida"> dá a descrição da quele imagem em <alt="descrição da imagem">.

<picture> <img src="exemplo de imagem:casa.png" alt="descrição:casa"> </picture>

Também para seu site trocar o tamanho da imagem conforme o tamango da tela do dipositivo, você precisa de pelomes três imagens de tamanhos diferentes,sendo eles 350px para celular 750px para tabletes e 1050px para computador e televisão.
Para poder fazer isso você utiliza a teg <source media type> como pode ver no exemplo a seguir:

<source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png">

Onde em mídia você coloca o tamanho máximo da imagem, e em <srcset="coloca a iamgem que você quer>.Para fazer que seu site diminua a imegem conforma a tela do usuário você utiliza o seguinte código:

<picture> <source media="(max-width: 750px)" srcset="imagens/foto-p.png" type="image/png"> <source media="(max-width: 1050px)" srcset="imagens/foto-m.png" type="image/png"> <img src="imagens/foto-g.png" alt="imagem flexível"> </picture>

No exemplo temos os seguints imagens imagens/foto-g.png imagem de tamanho grande,imagens/foto-m.png imagem de tamanho médio,imagens/foto-p.png imagem de tamanho pequeno.
Para fazer funcionar colocamos a teg <picture> dentro colocamos a teg <img> e colocamos a imagem grande, seguida colocamos a teg <source media=""> definimos o tamanho máxima da imagem grande, que é 1050px e colocamos em <srcset=""> a imagem pequena. Basicamente estamos falando que o tamanho máximo é 1050px e de caso a tela for menor que 1050px ele abrirar a imagem de tamnho médio,podemos ver essa mesma lógica na teg de cima, onde o tamanho máximo é 750px, se de caso a tela for menor abrirar a imagem pequena. assim seguindo uma sequência que a imagem maior fica em baixo depois a média e por ultimo a pequena.

Teg audio

Para adicionar áudio no seu site basta usar a teg <audio> e colocar o aúdio da sua escolha,mas para funcionar você tem que usar a seguintes tegs:

<audio src="" controls autoplay></audio>

O controls adiciona uma panel de controle onde você tem os controles básicos e o autoplay colocar para tocar sozinho. Formatos compativeis de áudio são eles: MP3 WAV OGG. obs: evite ao máximo usar o arquivo WAV porque ele é pesado.

Quando você colocar a teg <audio> você delata o src="".Em type você coloca o tipo de aquirvo, sendo que em mp3 o tipo de arquivo é você audio/mpeg.

Teg video

para arquivos de vídeos temos 3 formatos compatíveis MP4 M4V WEBM OGV, você pode usar teg <video> e dar enter que ele dará a opção de colocar o vídeo, mas a forma recomendada é abrir e fechar a teg <videos> exemplo:

<video></video>

Dentro dessa teg você coloca os vídeos que quer adicionar, lembrando que a ordem que você colocará será a ordem que o google vai ler, para você colocar os "botões" no vídeo basta colocar a teg controls entro da teg vídeo, exemplo:

<víde controls> </víde>

lembrando que para adicionar vídeos dentro da teg video, você utiliza a teg <source:src> você pode usar loop para deixar o vídeo rodando sem para e o autoplay para começar assim que entra na página, veja o exemplo:

<video> <video controls autoplay loop> <source src="exemplo de video.mp4" type="video/mp4"> </víde>

Você também pode adicionar uma thumb para seu vídeo basta colocar dentro da teg<video poster="coloca a imagem que vai ser a thumb"> assim colocando a imagem que você quer com thumb,veja o exemplo:

<video poster="coloca a imagem que vai ser a thumb">

Formulário

Criando sei formulário

Para criar um formulário utilizamos a teg form dentro dela colocamos um parágrafo e colocamos a opção a ser preenchida como: nome, endereo, número etc.. na frente de cada opção dentro do parágrafo colocamos a teg input-text e colocamos o nome e um id nessa teg, todo formulário tem o botão para enviar então criamos outro parágrafo e colocamos o imput-submit e escrevemos dentro das aspas na opção value="(Enviar)". Para você fazer que o parágrafo escrito nome tenham uma ligação com o campo onde você escreve, basta envelopar o text com a teg label e em for="" você coloca o nome do seu id do campo correspondente. Se você quiser tirar aquele caixa que aparece toda vez que clica em um campo do formulário basta ir na teg form e colocar autocomplete="off".A diferença entre name e id é que em php e html o name é mais aceito e o id é mais aceito em javascript.

get e post

No html existe duas maneiras de mandar dados de formulários que é o Get e Post, sendo que por padrão vem o get, a diferença entre eles é que o get manda os dados pelo a url do seu site e suporta um volume de dados menor não sendo recomentado para formulários grandes ou que orecisem mandar imagens e outros arquivos, já o post faz tudo que op get não faz tendo uma capacidade bem maior de enviaos de dados, mas quando eu devo utilizar os métodos o get e post? O get você utiliza quando seus dados não forem sensíveis como senhas e-mails entre outros dados, ele também não é adequado para envio de fotos, vídeos áudios e outros formatos de arquivos outra coisa a se avisar que o get só aceita enviar dados abaixo de três mil baytes acima disso ele não funciona. O método de Post consegue fazer tudo que foi dito que o get não consegue fazer.

Também pode se colocado um size para aumentar o tamanho da caixa, mas esse size é diferente o valor colocado será a quantidade de letras que é visualizada na caixa , isso não significa que ele limita a quantidade de letras que cabe lá, oque limita é o maxlength oque seria isso, quando você escreve um determinado número de caracteris ele pula para linha de baixo, oque o size faz é controlar esse limite.
Temos a opção de dar uma dica do que escrever dentro da caixa para isso utilizamos a teg placeholder="" dentro excrevemos a dica que queremos dá, como o campo é de endereço na dica colocamos a fraze escreva seu endereço aqui.

Para fazer uma media quewry para impressora criamos um link css e dentro do link colocamos a teg media="print" tem como definir a media query da tela do seu site e basicamente o link css que você fazia só com media="screem" no final. A media query é igual a media <type = media filtre>, Você pode fazer um css para telas em modo retrato ou em paisagem, basta usar no link css media="screen and (orientation portrait)" retrato e media="screen and (orientation landscape)" paisagem, o nome dessa configuração é media filtre que é basicamente uma característica do media type. Se você quiser todas as configurações de media no mesmo css, basta usar a declaraão @media screen and (orientation:portrait){} e @media screen and (orientation:landscape){} dentro do coxete colocamos as configurações que desejamos para cada media types, lembrando quando você colocar todas as declarações de media dentro de um css, não precisa colocar a teg media dentro do link css.
