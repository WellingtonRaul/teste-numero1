Repositório local e Remoto

Git é um sowfter que guardar versões anteriores do seu site,podendo ser chamado de repositório local.

O Github é uma plataforma como uma rede social de programadores, nele você pode postar seus projetos podendo ser visto por empresas, nela também podemos hospedar seu site de forma gratuita, ele é chamdo de repositório remoto.
