Teg Global *

serve para fazer uma configuração global do seu css, nele você coloca, como exemplo dexar seu site com padding 10px margin de 10px entre outras configurações.

Teg Main Footer e Header

A teg main significa que tudo que está dentro dela é o conteúdo principal assim podendo separar-la para configuarar. A teg footer é o rodapé do site, nela você pode colocar como exemplo o nome de quem criou o site. A teg header, se um temos temos um roda pé também temos o cabeçalho, nela você pode colocar o título botões interativos barras de pesquisas entre outras opções.

Tegs Width Padding e Margin

width serve para definir a largura seja das caixas do seun site entre outos usos. A teg padding é o espaçamento em relação a borda a borda, serve para deixar mais espaçado os seus texto, vídeos, aúdios entre outros.

Box-shodow

A teg box-shadow serve para colocar sombras nas boxs,veja um exemplo <box-shadow: 10px 10px 10px 10px 10px black;> a sequência dos 10px cada um significa uma direção, dada na seguinte sequência: cima direita baixo esquerda, o quinto da sequência e o espalhamento da sombra, e black é a cor da sombra, você também pode colocar um valor ele fará que todas as outras direções tenham mesmo valor, também você pode colocar dois valores, sendo o primeirop valor definindo cima e baixo e o segundo valor definindo direita e esquerda.
Clicando na cor que você adicionou você pode trocar ela para rbg e rgba e também mudar sua transparencia isso serve para quando você adicionar cores em qualquer teg.

Safe Combinations

Quando for colocar fontes no seu site tem que levar em conta que existem alguns aparelhos que não existem aquela fonte, sendo as mais utilizadas Arial, Helvetica, sans-serif quando você coloca várias opções de fontes o google irá utilizar a primeira e se não reconhecer irá utilizar a segunda e assim por diante. Esse tipo de combinação é chamada de safe combination são combinações de fontes normalmente parecidas que pode ter em todos os aparelhos. O visual studio code já dá safe combinations mas ele não mostra todos que existem. Para colocar uma fonte é sempre bom saber a altura do corpo da fonte.

Fonte Serifadas

Fontes sarifadas são aquelas que possuem "traços em sua base como perninhas".Se colocar em <font-Family=""> e escrever sans-serif, serif, monospace ele colocará qualquer fonte sem serifa com serifa ou mono espaçada que ele achar.

Medidas Absolutas e Peso

No temos medidas absolutas e relativas, nas medidas absolutas temos cm, mm, in, px, pt e pc, como o nome diz elas são medidas que representam exatamente o número a qual foi definido. As medidas relativas são elas: em, ex, rem, vlw, vh, e %, essas medidas são relativas ao tipo de medida que foi atribuída á ela. Pelo w3c é recomendado utilizar as medidas px e em. As fontes tem pesos isso define se a fonte vai ser mais fino ou mais espeça a teg utilizada para colocar peso na fonte é existem três pesos mas aceitos são eles: loghter, normal, bol e bolder, outra forma de definir o poso basta usar números que vão de 100 á 900.

Itálico e Negrito

Você pode usar teg font-style: ; para definir se o texto será em itálico negrito ou sublinhado, podemos colocar em sublinhado utilizando a teg <text-decoration:underline;> um ponto interessante a ser relevar é que nem toda Fontes permitem tantas configurações então preste atenção.

Shorthend Das Tegs Font

Para simplificar essas configurações podemos utilizar um shorthand font que basicamente serve para condensar todas as configurações em um código, assim utilizando a teg font e colocando as configuração em uma sequência, sendo ela: o style primeiro weight segundo siz e terceiro e por ultimo Family, nessa sequência você não é obrigado a colocar todas configurações por exemplo não quero definir o meu style é só não colocar nada e continuar a sequência.

Fontes Externas

para colocar uma fontes externa tem criar uma regra chamada @font-face ela dará várias opções em src: url () você coloca o nome do arquivo da fonte depois do parênteses você coloca format() nele você coloca uma determinada opção em relação ao tipo de formato da fonte, exemplo: ser otf você coloca opentype ser for ttf coloca e ser for truetype-att você oloca apple advanced typography, também tomos o svg.

Alinhamento de textos

Para alinhar o texto utilizamos a teg text-align ai definimos centro esquerda direita e justifica que alinha com a esquerda e a direita, também temos a teg text-indent onde damos aquela espaçamento no começo do texto, aquele que aprendemos na escola.

Class e Id

De caso você tenha vários h1 e um em especifico você quer que tenha uma coinfiguração diferente basta colocar um identificador utilizando dentro da teg que você quer modificar ,exemplo: <h1 id="coloque aqui o nome do identificador">, para colocar esse identificador dentro do css você coloca (h1# aqui o nome do identificardor) em css você não precisa colocar a teg que aquele identificador referencia, basta colocar # e o nome do identificador. Outro ponto interessante é que o indetificador em html é id já em css é #. Dentro do html você não pode usar o mesmo id mais de uma vez.
As class difentes dos indetificador você pode usar quantas vezes quiser e também atribuir várias classes para a mesma teg, também podendo colocar class e id juntos. Para colocar classe basta abrir a teg e colocar na frente, exemplo: <h1 class="aqui coloca o nome da class que você deseixar" >. As clas em css são representas como ponto, exemplo: (h1 .nome de class {}).

Pseudos Classes

As pseudos classes serve para adicionar uma característica especial em elementos e classe, exemplo se você quer que um determinado texto esteja invisível e só seja visto quando passa o mouse por cima você utiliza pseudos classes, veja o exemplo: 

                
                    div:hover{
                        color: red;
                    }
                     div > p {
                        display: none;
                    }
                    div:hover > p {
                        display: block;
                        color: white;
                        background: red;
                    }
